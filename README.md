# hackerthon-django

## Requirements

#### Product

- Python (3.6)
- pipenv
- Django (2.x)

#### Dev

- ipython
- Django-Extensions

### Secrets

#### `.secrets/base.json`

```json
{
  "SECRET_KEY": "<Django secret key>"
}
```

## Running

```
# Move project's directory
pipenv install
pipenv shell
./runlocal.sh
```
